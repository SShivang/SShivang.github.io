College Counselling Should Not Be So Hard

EduGuide is a platform that uses machine learning to give recommendations on key decisions in the college admissions process such as which colleges to apply to and what extracurriculars to take part in.

Login and Signup Pages are fully developed

Using Google firebase, the EduGuide team has configured secure user authentication.

Surveys and Data collection

Using surveys and cursory data we are able to optimize our alogrithms to provide you with the best recommendations.
